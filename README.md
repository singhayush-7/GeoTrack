#GeoTrack  
It is a device location mapping tool that tracks and displays real-time locations of devices.
## 🚀 Features
- Displays device location on map  
- Real-time tracking
- shows history
 
- ## 🛠️ Tech Stack 
- TailwindCSS  
- Socket.IO
- JavaScript
- MERN
